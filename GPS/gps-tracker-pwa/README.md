# Bus Tracker MVP (PWA)

PWA para rastreo de buses con GPS, detección de paradas, Firebase (Firestore) y mapa con Leaflet.

## Configuración antes de ejecutar

1. Copia `.env.example` a `.env` en esta carpeta.
2. Crea un proyecto en [Firebase Console](https://console.firebase.google.com) y configura Firestore.
3. Rellena las variables en `.env` con los valores de tu proyecto (Configuración del proyecto → Tus apps → SDK).

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — desarrollo
- `npm run build` — build para producción
- `npm run preview` — previsualizar build
- `npm run lint` — ESLint

```js
      // Alternatively, use this for stricter rules

      // Optionally, add this for stylistic rules
      tseslint.configs.stylisticTypeChecked,

      // Other configs...
    ],

      parserOptions: {
        project: ['./tsconfig.node.json', './tsconfig.app.json'],
        tsconfigRootDir: import.meta.dirname,
      },
      // other options...
    },
  },
])
```

You can also install [eslint-plugin-react-x](https://github.com/Rel1cx/eslint-react/tree/main/packages/plugins/eslint-plugin-react-x) and [eslint-plugin-react-dom](https://github.com/Rel1cx/eslint-react/tree/main/packages/plugins/eslint-plugin-react-dom) for React-specific lint rules:

```js
// eslint.config.js

import reactDom from 'eslint-plugin-react-dom'

export default defineConfig([
  globalIgnores(['dist']),
  {
    files: ['**/*.{ts,tsx}'],
    extends: [

      // Enable lint rules for React

      // Enable lint rules for React DOM
      reactDom.configs.recommended,

    languageOptions: {
      parserOptions: {
        project: ['./tsconfig.node.json', './tsconfig.app.json'],
        tsconfigRootDir: import.meta.dirname,
      },
      // other options...
    },
  },
])
```
